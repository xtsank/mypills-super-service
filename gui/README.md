# MyPills GUI

Минимальный GUI для сервиса MyPills на React + TypeScript с MVVM-структурой.

## Быстрый старт

```bash
npm install
npm run dev
```

## Переменные окружения

- `VITE_API_BASE_URL` — базовый URL API (например, `http://localhost:8080`).

## Скрипты

- `npm run dev` — дев-сервер Vite.
- `npm run build` — сборка.
- `npm run preview` — предпросмотр сборки.
